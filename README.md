### Setup:
> run `npm install` to install all the dependencies
```shell
npm install
```

> This project uses a `.env` file
The `.env` variables you will need to supply are
```shell
VITE_APP_MY_API_LINK=
```

### Run:
> start the application
```shell
npm run dev
```
